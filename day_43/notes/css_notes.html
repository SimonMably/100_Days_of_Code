Some CSS have default values CSS Default Value Reference.

Internal CSS is CSS code inside a HTML file (specifically in the head section). External CSS is CSS code in its own CSS file and is linked to within the head section of a HTML file. For an entire website, a CSS file needs to be linked to within each web page.

Use Use tag selectors to change the properties of all instances of a particular tag. Use class selectors to change the properties of a single/particular tag. Use the class property inside of the tag first, then change the property in CSS. click here for more information on the tag selectors and class selectors. Can also use id selectors. This will overide tag selectors.
